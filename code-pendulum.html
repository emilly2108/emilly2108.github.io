도립진자 강화학습 제어 프로젝트

2026.06

RoboMaster S1과 Jetson Orin Nano를 활용해 도립진자 강화학습 알고리즘과 제어 흐름을 실험한 프로젝트입니다.

이 프로젝트는 도립진자 문제를 로봇 제어 관점에서 다루며, 강화학습 알고리즘을 실제 장치와 연결하는 흐름을 실험하기 위해 정리한 코드입니다. GitHub 기록 기준 저장소는 2026년 6월 25일에 공개되었습니다.

  • 작업 시기 2026년 6월
  • 프로젝트 성격 도립진자 강화학습 알고리즘과 로봇 제어 실험
  • 주요 장비 RoboMaster S1, Jetson Orin Nano

프로젝트 내용

도립진자는 균형을 잃기 쉬운 시스템을 제어해야 하므로 강화학습과 로봇 제어를 연결해 보기 좋은 문제입니다. 이 프로젝트에서는 로봇이 현재 상태를 바탕으로 어떤 제어 입력을 선택해야 하는지 실험하는 데 초점을 두었습니다.

핵심 포인트

  • 도립진자 균형 제어를 위한 강화학습 알고리즘 실험
  • RoboMaster S1과 Jetson Orin Nano 기반 제어 흐름 구성
  • 센서 입력, 제어 명령, 학습 정책을 연결하는 구조 탐색
  • 로봇 강화학습을 실제 하드웨어 문제로 확장하기 위한 기반 코드 정리

이 프로젝트는 이후 더 복잡한 로봇 강화학습 실험으로 넘어가기 위한 출발점 역할을 합니다. 단순한 예제가 아니라 실제 장치와 제어 흐름을 엮어 보며, 학습 정책이 물리 시스템에 적용될 때 어떤 점을 고려해야 하는지 확인하는 데 의미가 있습니다.
