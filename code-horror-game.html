유니티를 활용한 공포게임 협업 작품

2025.07–2025.08

2025년에 약 1개월간 친구와 함께 제작한 첫 협업 작품입니다. 공포게임 안에서 플레이어가 퍼즐을 풀고, 사물과 상호작용하며, 애니메이션으로 상황 변화를 체감하도록 구현했습니다.

이 프로젝트는 Unity를 활용해 만든 3D 공포게임 협업 작품입니다. GitHub 기록 기준 저장소는 2025년 7월 14일에 생성되었고, 마지막 주요 작업은 2025년 8월 18일에 반영되어 약 한 달 동안 집중적으로 제작했습니다.

  • 작업 기간 2025년 7월 14일 ~ 2025년 8월 18일
  • 협업 방식 친구와 함께 역할을 나누어 제작한 Unity 공포게임 프로젝트
  • 주요 기술 Unity, C#, ShaderLab, HLSL

내가 담당한 작업

  • 게임 안의 모든 퍼즐 로직 코딩
  • 문, 오브젝트, 아이템 등 사물과의 객체 상호작용 코딩
  • 사물 상태 변화에 맞춘 애니메이션 동작 코딩
  • 플레이어 행동에 따라 퍼즐과 오브젝트가 자연스럽게 연결되도록 흐름 구현

프로젝트에서 중요하게 본 점

공포게임은 분위기만큼이나 플레이어가 “무엇을 눌렀을 때 무엇이 변하는지”를 자연스럽게 이해하는 흐름이 중요합니다. 그래서 퍼즐의 조건, 사물과의 상호작용, 애니메이션 재생 타이밍이 끊기지 않도록 연결하는 데 집중했습니다.

이 작업을 통해 Unity에서 상호작용 오브젝트를 설계하고, 게임 플레이 흐름에 맞춰 스크립트를 나누어 관리하는 경험을 쌓았습니다. 첫 협업 작품이라 기능 구현뿐 아니라 작업 분담과 결과물 통합 과정도 함께 배울 수 있었던 프로젝트입니다.
